🍃 Reactを用いたポートフォリオサイト

概要

このプロジェクトは、React を用いて開発したポートフォリオサイトです。
自己紹介や来歴、制作物、お問合せ先などの紹介を備えています。

🛠 使用技術・サービス

  • React
  • AWS (公開に使用 | EC2, Route53など)
